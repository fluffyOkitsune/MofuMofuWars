# もふもふうぉーず

## 概要
おきつねとおおかみが戦う、もふもふでかわいい戦闘シュミレーションゲーム。

##システム
味方が全滅するとゲームオーバー。

### レベルアップ
ステージをクリアすると参戦したキャラクターはレベルアップする。
撃破されなかったキャラクターのうち、戦闘回数が多いキャラクターは追加でレベルアップする。

### 武器の習熟度
戦闘をすると、ダメージを与えたか否かにかかわらず経験値がたまり、一定値を超えると武器の習熟度が上がる。武器の習熟度が5になると、上のランクの兵種になり、新たな種類の武器が使えるようになる。

### モフる
キャラクターが二人並んだ状態で攻撃を行うと発生。攻撃を行った際、攻撃をしていない味方が相手をモフり、反撃を防ぐ。ただし、モフる対象のキャラクターに隣接したキャラクターが居るとき、モフりはするがモフらなかったほうの相手から威力半分の反撃を受ける。

### 月齢
８ターン周期で月が満ち欠けし、満月になると与えるダメージと受けるダメージが10増え、新月になると半減する。

## 戦闘システム
### キャラクターのパラメータ
* 力	：物理攻撃力の基礎値
* 魔力	：魔法攻撃力の基礎値
* 防御	：相手の物理攻撃ダメージを低減する
* 魔防	：相手の魔法攻撃ダメージを低減する
* 器用さ：攻撃の命中率
* 回避率：相手の攻撃を回避できる確率
* もふ度：コマンド『モフる』の成功率
* 武器の習熟度：扱える武器のレベル

## 武器
* 槍	：特別な効果が発生しないスタンダードな武器。
* 剣	：追撃ができる。剣同士では交互に二回ずつ攻撃することになる。必殺なし。  
* 斧	：自分から攻撃すると与えるダメージが増えるが、相手から攻撃されると受けるダメージが増える。  
* 魔術書：魔法で攻撃する。
* 呪い	：ステータスを変化させるなどの効果を与える。
* 杖	：癒しの力で体力を回復する。
* 弓	：定まった射程のマスを攻撃できるが、隣接マスからの攻撃はダメージ２倍になる。
* 暗器	：必殺率が高いが、攻撃力が低い。
* 銃	：威力、必殺率が高いが攻撃すると次のターン攻撃できない。

## 兵種
### おきつねの兵種
おきつねは呪いで相手を撹乱し、暗記や弓で戦うテクニカルな陣営。
* 侍	：剣が装備できる。回避率が高く、連続攻撃で体力を奪う。
* 夜叉	：剣・槍が装備できる。器用さが高く、回避率が非常に高い。

* 豪傑	：斧が装備できる。力が高く、硬い相手も攻撃できる。
* 修羅	：斧・槍が装備できる。器用さが高く、力が非常に高い。

* 巫女	：杖で味方を回復する。　攻撃はできない。

* 邪術師：呪いで相手の行動を邪魔したり、味方をサポートする。攻撃はできない。
* 陰陽師：呪いに加え、魔術で相手を攻撃する。

* 忍者	：暗器を装備でき、相手を抹殺できる。忍者きたない。

* 猟師	：銃を装備できる。パラメーターが全体的に低い。

### おおかみの兵種
おおかみは高いステータスで殲滅できる優秀な陣営。
* グラディエーター：剣が装備できる。ステータスのバランスが良い。
* ヴァルキリー：剣・斧が装備できる。

* ランサー：槍が装備できる。力と防御力に優れる。

* シスター：杖で味方を回復する。　攻撃はできない。

* マージ：魔術書が装備できる。魔力と魔防に優れる。
* ソーサラー：魔術書が装備できる。魔力が非常に高い。呪いも多少は扱える。

* アサシン：暗器を装備できる。防御力が劣る。

* ガンナー：暗器と銃を装備できる。防御力がやや劣る。
